# Random Password Generator

## Description
A Python application that generates strong and secure passwords using letters, numbers, and special characters.

## Features
- Custom password length
- Secure password generation
- Includes symbols and numbers

## Technologies Used
- Python
- string module
- secrets module

## How to Run
1. Open terminal
2. Run:
   python password_generator.py

## Sample Output
Enter password length: 12
Generated Password: A#7kP!2mX@9q
